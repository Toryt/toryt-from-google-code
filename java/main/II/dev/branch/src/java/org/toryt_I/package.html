Compatibility package. This package contains code that enables Toryt II to work with Toryt I {@link org.toryt_I.Contract Contracts}.

To use Toryt I contracts with Toryt II, include both the Toryt I jar and the Toryt II jar in the classpath.

In Toryt II, the interface CaseProvider is no longer used. Existing contracts could safely remove dependencies on this interface, and implementations of the methods of these interfaces.

In Toryt II, method contracts find out there cases themselves. The getCases() methods are not called, but they have to be in the contract for compatibility with the Toryt I interface.
